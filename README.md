# CloudCostLens

CloudCostLens is a data engineering and analytics engineering portfolio project for cloud cost observability.

The project generates realistic cloud billing data, loads it into Snowflake, transforms raw usage records with dbt, detects cost anomalies, tracks budget performance, and visualizes cloud spend trends in a Streamlit dashboard.

## Why I Built This

Cloud costs are a major operational concern for modern data and software teams. I built CloudCostLens to practice cloud data warehouse design, ELT pipelines, dbt modeling, orchestration, and business-facing analytics around cost monitoring.

## Planned Tech Stack

- Python

- Snowflake

- dbt

- Airflow

- Streamlit

- SQL

- GitHub Actions

## Planned Features

- Generate realistic cloud billing data

- Load raw billing data into Snowflake

- Build dbt staging and mart models

- Track daily spend by service, project, environment, and team

- Detect cost spikes and budget overruns

- Visualize spend trends and anomaly alerts

- Orchestrate the workflow with Airflow

- Add GitHub Actions CI
